# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-08

### Added
- Initial release
- Autonomous email generation loop (Karpathy autoresearch pattern)
- 5 execution modes: autonomous, preview, resume, single contact, approval
- TSV-based contact tracking with deduplication
- Lead status-based tone and greeting selection
- Context extraction from HubSpot notes
- Gmail draft creation via MCP
- Configurable skip flags
- Error logging to output/errors.log
